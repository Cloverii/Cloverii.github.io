之后就可以 init, generate, server 啦！

补充关于站点配置文件里language的写法：对于maupassant主题，应该写 zh-CN

一开始查了List of ISO 639-1 codes之后我直接写了zh，站点依然是英文且偶有乱码；

后来看到 NexT 文档里的简体中文写法是zh-Hans，就改了下，依然英文+乱码；

后来又查到了这个，改成 zh-CN，终于OK了。
